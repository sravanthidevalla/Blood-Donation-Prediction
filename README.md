# Blood Donation Prediction Using Machine Learning

## Project Overview

This project aims to predict whether a person is likely to donate blood in the future based on their historical donation records. Machine learning techniques are used to analyze donor behavior and build a predictive model that can help blood banks identify potential donors.

## Problem Statement

Blood banks often face challenges in maintaining an adequate blood supply. Predicting potential donors can help organizations conduct targeted donation campaigns and improve blood inventory management.

## Dataset Features

* Recency (months since last donation)
* Frequency (total number of donations)
* Monetary (total blood donated)
* Time (months since first donation)

## Evaluation Metrics

* Accuracy Score
* Precision
* Recall
* F1 Score
* Confusion Matrix

## Project Outcome

The developed model successfully predicts the likelihood of future blood donation based on donor history. This can assist blood banks in identifying and targeting potential donors more effectively.
